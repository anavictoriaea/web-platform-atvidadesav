Atividade 4 - Menu

Bem-vindo!

Olá! Esta página que você está vendo agora serve para guiar você até as outras páginas desta atividade, a partir do menu acima.

Diferente dos outros exercícios que fiz, esta atividade não traz nada de novo além do design. Espero que goste deste visual.

Para montá-lo, fiz uso de Cascading Style Sheets (CSS), que é uma linguagem usada para criar folhas de estilo. O gradiente usado na página de fundo foi obtido através da ferramenta de paletas Coolors.co (clique no link para conferir a paleta). A fonte usada neste texto e em todas as páginas é a "Trebuchet MS" e suas variações. Você pode acessar o meu Github no fim da página ou meu perfil no Instagram na página Contato.

Acredito que valha a pena compartilhar músicas que estive ouvindo enquanto digitava. Pois bem, o gênero musical da vez é "electronica", recomendo as músicas do Mareux e do duo Crystal Castles.
